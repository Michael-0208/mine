
# Mine Portfolio Website

Dark-themed personal portfolio built with React, TypeScript, Vite, and Tailwind CSS.

## Overview

This project contains:
- A landing page with sections like Hero, About, Expertise, Services, Tech Stack, and Contact.
- A portfolio listing page.
- A case study page for **UpLevo POS**.

## Tech Stack

- React + TypeScript
- Vite
- React Router
- Tailwind CSS
- Motion (`motion/react`) for animations
- Radix UI-based components

## Routes

- `/` - Home page
- `/portfolio` - Featured projects
- `/portfolio/uplevo-pos` - UpLevo POS case study

## Getting Started

1. Install dependencies:

```bash
npm install
```

2. Run development server:

```bash
npm run dev
```

3. Build for production:

```bash
npm run build
```

## Notes

- This is a Vite SPA; ensure Node.js is installed before running.
- The app starts with a dark theme and smooth scrolling enabled in the root app setup.
